Meu Site Flexbox

Card 1

Este é o primeiro card. Usando Flexbox, ele se alinha automaticamente ao lado dos outros, crescendo ou encolhendo para preencher o espaço de forma inteligente.

Card 2

O segundo card. Note como todos os cards têm a mesma altura, mesmo com quantidades diferentes de texto. O Flexbox cuida disso para nós!

Card 3

O terceiro card. Em telas menores, se não houver espaço, podemos configurar o container para "quebrar" a linha (flex-wrap), e este card iria para baixo.
